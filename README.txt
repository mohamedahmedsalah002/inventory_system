Inventory System - Full (Bootstrap, CSS, no-login)
-------------------------------------------------
Installation (mac + XAMPP):
1. Start XAMPP - Apache & MySQL.
2. Copy the folder 'inventory_system_full' into:
   /Applications/XAMPP/xamppfiles/htdocs/
3. Open phpMyAdmin: http://localhost/phpmyadmin
4. Create database 'inventory_system' or import inventory.sql directly into a new DB.
5. Import the file inventory.sql (inside the folder).
6. Open the app in browser: http://localhost/inventory_system_full/index.php

DB settings are in db.php (default root, empty password for XAMPP).
Debug mode is ON (error display). Remove error display for production.
